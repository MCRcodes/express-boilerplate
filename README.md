# express-boilerplate

> Simple Express Backend w/ Webpack, ES6, ESLint, Hot-Reload Template

```bash
git clone https://github.com/chryb/express-boilerplate

cd express-boilerplate

# make it to your own
rm -rf .git

npm install

# hot reload
npm run dev

# build
npm run build

# build & start
npm start

# test
npm run unit
```
